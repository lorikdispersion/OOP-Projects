Добро пожаловать в RayTracer

 Эта программа написана для создания изображений
 при помощи технологии Ray Tracing 
https://ru.wikipedia.org/wiki/%D0%A2%D1%80%D0%B0%D1%81%D1%81%D0%B8%D1%80%D0%BE%D0%B2%D0%BA%D0%B0_%D0%BB%D1%83%D1%87%D0%B5%D0%B9

На данный момент реализовано оторбажение неограниченного количества шаров разных радиусов и разных центров из 3-х различных материалов:

-Матовая поверхность на основе закона Ламберта 
https://en.wikipedia.org/wiki/Lambert%27s_cosine_law ;

-Металлическая поверхность с симуляцией отражения лучей и параметром неровности поверхности, искажающей отражения;

-Диэлектрический (прозрачный) материал, симулирующий преломление лучей на основании закона Снеллиуса 
и отражение лучей, падающих под крутым углом в соответствие с аппроксимацией Шлика
https://en.wikipedia.org/wiki/Schlick%27s_approximation;

При запуске этой программы открывается консоль и от пользователя требуется ввести последовательно следующие параметры 
(подсказка, какой параметр нужно ввести прям сейчас выводится прямо в консоль):

-Путь выходного файла, строго формата .ppm (например output.ppm);

-Ширина изображения (например 1200);

-Длина изображений (например 800);

-Положение камеры в трехмерном пространстве (например 1, -1, 0.5)

-Точка, в которую смотрит камера в трехмерном пространстве (например 0, -0.3, 1)

-Направление на верх камеры в трехмерном пространстве (например 0, 1, 0)

-Поле зрение камеры в градусах (например 90)

-Фокусное расстрояние камеры (например 10)

-Апертура камеры (например 10)

-Степень гамма-коррекции (например 0.5)

-Глубина - максимальное число отражений на каждый луч (например 100)

-Количество лучей, испускаемых на каждый пиксель - влияет на качество картинки (например 200)
https://en.wikipedia.org/wiki/Spatial_anti-aliasing

Затем наступает блок инициализации сцены. От пользователя требуется указать число объектов, которые вы хотите видеть на итоговом изображении.
После, нужно описать тип материала объекта и сам объекта. Ориентируйтесь на подсказки.

Пример блока инициализации сцены:
4

Lambertian 0.8 0.8 0.0
Sphere 0.0 -100.5 -1.0 100.0

Lambertian 0.1 0.2 0.5
Sphere 0.0 0.0 -1.0 0.5

Dielectric 1.5
Sphere -1 0.0 -1.0 0.5

Metal 0.8 0.6 0.2 0.1
Sphere 1 0.0 -1.0 0.5


После этого в консоль выводится шкала прогресса обработки изображения. Рисунок можно увидеть только по истечении работы программы.

Файл формата .ppm можно открыть при помощи некоторых обработчиков изображений (например GIMP)
или при помощи интернет сервисов https://www.cs.rhodes.edu/welshc/COMP141_F16/ppmReader.html
